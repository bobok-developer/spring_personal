## 🎈 설명

- 부트스트랩4 기반 풀페이지 형식의 사이트입니다.
- 진료 예약 기능과 회원 가입 시 아이디 중복 확인은 시간상 구현하지 못 했습니다.

### 🍍 기능

1. 회원
- 회원가입, 로그인, 내 정보 수정, 회원 탈퇴
<br>

2. 게시판
- 게시글 CRUD- 비밀글 작성 가능, 게시글 목록에 대한 페이징 처리, 키워드로 검색 가능
- 파일첨부 (Ajax)
- 게시글에 대한 댓글 CRUD (Ajax)
<br>

3. OAuth2.0 카카오 로그인 구현 (2021. 08. 17. 추가)
